# LogEstoque

Projeto de Desenvolvimento de Software para curso de Engenharia de Software.

O projeto consiste em uma aplicação de movimentação de material por depósito e centro utilizando o framework [SAP CAP](https://cap.cloud.sap/docs/) e banco de daods [Postgres](https://www.postgresql.org/).

⚠️ Important Notice: This is an educational and learning project.

SAP Cloud Application Programming (SAP CAP) is a proprietary technology and framework developed and patented by SAP SE. This repository is not an official SAP product and has no affiliation, sponsorship, or endorsement from SAP.

The purpose of this project is to serve as educational material for developers interested in understanding SAP CAP concepts, patterns, and best practices. All examples, code, and documentation herein were developed independently for educational purposes.
